Zeta Code Help

About

Zeta Code is a way to view code as a graph. It does not mean to replace traditional IDE's, but rather is a different way to view and manipulate code. It looks at a project holistically - including imports and dependencies as top-level objects and breaks out of the ancient code-as-files metaphor.

Supports

  • Java and Maven are directly supported
  • ...API for additional languages and frameworks

Getting Started

  • Start Zeta Code
  • Right-click anywhere - this shows the menu
  • Click File-->Open
  • Open an existing Maven-based Java project (directory or pom file).

Navigating around

  • Left-click on a node to select it.
  • To go up a level at any time: choose "Go up" in the menu.
  • Ctrl-click on a node to edit it.

Editing Files

  • Your changes are saved every time you hit Enter
  • Additional help is available by hitting F1 in the editor.
  • Esc - exit the editor
